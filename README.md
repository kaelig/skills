# skills

Personal agent skills repository.

## Install all skills

```sh
npx skills add kaelig/skills
```

Or download individual `.skill` files from this repo and open them in Claude Desktop.

## Skills

| Skill | Install | Description |
|-------|---------|-------------|
| [company-evaluator](company-evaluator/) | `npx skills add kaelig/skills --skill company-evaluator` | Produce a structured employer evaluation report for a specific company |
| [job-search-refresher](job-search-refresher/) | `npx skills add kaelig/skills --skill job-search-refresher` | Sweep the design systems job market for new Staff/Principal roles, deduped against past searches |
